Interactive Tableau dashboard exploring 9,000+ movies & TV shows  
Global distribution • Top genres • Ratings • Release trends

**Live Interactive Dashboard → [Explore on Tableau Public](https://public.tableau.com/app/profile/thao.pham7533/viz/AmazonDashboard_17435561739030/AmazonDashboard)

### Key Insights
- United States dominates content production (2,500+ titles)
- Drama & Comedy are the top genres
- 80.82% Movies vs 19.18% TV Shows
- Highest rated content: 13+ and 16+ categories
- Massive growth in original content post-2015
